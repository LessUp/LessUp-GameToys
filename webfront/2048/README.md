# 2048 游戏

纯前端单文件实现的 2048 数字合并游戏，支持键盘与触摸手势操作。

## 特性
- 纯 HTML/CSS/JavaScript，无需构建工具
- 响应式设计，支持桌面与移动端
- 键盘控制（方向键、WASD）
- 触摸手势支持（滑动操作）
- 实时得分显示
- 游戏结束提示
- 平滑动画与渐变色彩

## 使用方式
直接在浏览器中打开 `index.html` 即可开始游戏。

## 操作说明
- **桌面端**：使用方向键（↑ ↓ ← →）或 WASD 控制
- **移动端**：在游戏区域滑动手指控制方向
- **新游戏**：点击右上角的"新游戏"按钮

## 游戏规则
- 通过方向键移动方块
- 相同数字的方块相撞会合并为一个数字翻倍的新方块
- 每次移动后会随机生成一个新方块（2 或 4）
- 目标是合成 2048 方块
- 无法移动时游戏结束
