# 双摆混沌系统

> 经典双摆物理系统，展示混沌运动与轨迹追踪。

## 功能亮点
- Canvas 实时渲染双摆轨迹，可切换显示尾迹长度与渐隐效果。
- 参数面板支持调节摆长、摆锤质量、重力、阻尼等物理量，实时观察混沌响应。
- 预设初始角度输入框，方便快速复现特定轨迹或对比多个实验场景。

## 快速开始
1. 在浏览器中直接打开 `index.html`。
2. 通过右侧滑块调整参数，或点击“重置初始化”按钮恢复默认值。
3. 直接拖拽末端质点可重设初始角度，双击画布可快速清除轨迹。

## 交互提示
- 勾选“轨迹”相关选项可以开启/关闭轨迹绘制，调整尾迹长度可改变历史留存时间。
- 当参数导致系统失稳时，可使用“暂停/继续”按钮观察某一帧的姿态。

## 相关文件
- `index.html` —— 主实现文件，包含样式、控制面板与物理求解逻辑。
- `meta.json` —— 模块元数据，供首页卡片与自动化脚本使用。
