# 粒子爆炸特效

> 鼠标交互触发的烟花、爆炸与粒子轨迹效果。

## 功能亮点
- 多主题粒子着色方案（极光 / 余烬 / 深海 / 赛博），支持即时切换。
- 控制面板可调节自动触发间隔、粒子数量、尾迹长度与重力方向。
- 支持多种爆炸模式（烟花、能量脉冲、冲击环、火花雨），并显示实时 FPS 与粒子存活统计。

## 快速开始
1. 在浏览器中打开 `index.html`。
2. 使用控制面板调整参数或选择主题、爆炸模式。
3. 点击画布任意位置触发特效；如需手动测试，可使用“立即爆炸”按钮。

## 交互提示
- 长按画布会持续喷射粒子，双指缩放可在触屏设备上调整全局缩放。
- “暂停自动”可暂时关闭定时触发，配合“清除残留”快速观察新一轮效果。

## 相关文件
- `index.html` —— 主实现，包含粒子系统逻辑与交互面板。
- `meta.json` —— 模块元数据，供首页自动化脚本引用。
