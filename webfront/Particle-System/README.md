# 粒子系统引擎

> 可配置的粒子系统，支持重力、风力、摩擦力等多种力场。

## 功能亮点
- 发射速率、寿命、重力、风力、摩擦、噪声等关键参数均可通过滑块实时调整。
- 提供扇形喷射、中心爆裂、旋转环和平行流等发射模式，并内置多套色彩方案。
- 支持尾迹长度控制、爆炸触发、全局暂停及清屏，便于调试不同组合效果。

## 快速开始
1. 在浏览器中打开 `index.html`。
2. 通过控制面板调节粒子行为或切换发射模式、色彩方案。
3. 点击“触发爆炸”体验瞬时爆裂效果，或使用“清除”按钮重新开始。

## 交互提示
- 在画布按下并拖动可重新定位发射器，并改变粒子喷射方向；滚轮可缩放观察区域。
- 若性能压力过大，可降低“最大粒子数”或缩短寿命以保持流畅。

## 相关文件
- `index.html` —— 粒子求解、渲染与控制面板全部集中于此。
- `meta.json` —— 模块元数据，供首页自动生成卡片使用。
