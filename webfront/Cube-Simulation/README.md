# 3×3 魔方模拟器

> 完整的 3D 魔方，支持各种旋转操作、打乱与视角控制。

## 功能亮点
- 内置 U/D/L/R/F/B 等基础旋转按钮，并提供一键打乱、重置功能。
- CSS 3D 渲染魔方外观，拖拽、滚轮与双击分别用于旋转、缩放与复位视角。
- 支持动画过渡与状态同步，连续操作时仍保持平滑体验。

## 快速开始
1. 在浏览器中打开 `index.html`。
2. 使用上方按钮执行基础转动，或点击“打乱”生成随机序列。
3. 拖拽场景旋转视角，配合滚轮缩放查看不同角度细节。

## 交互提示
- 双击场景可快速回到默认视角，适合重置观察角度。
- 操作过程中可随时点击“重置”恢复初始状态。

## 相关文件
- `index.html` —— 魔方建模、动画与交互的核心实现。
- `meta.json` —— 模块元数据，供首页和生成脚本读取。
