# 弹簧物理系统

> 可交互的弹簧-质点系统，模拟布料与软体动力学。

## 功能亮点
- 滑块调节布料尺寸、重力、弹性、阻尼与风力等核心参数。
- 支持不同的固定点方案与弯曲约束，可快速切换布料拓扑。
- 实时统计节点/弹簧数量与平均速度，便于分析性能与动态表现。

## 快速开始
1. 在浏览器中打开 `index.html`。
2. 通过面板调整重力、风力或弯曲模式，观察布料响应。
3. 使用“瞬间风暴”制造突发外力，或点击“重置布料”恢复初始状态。

## 交互提示
- 拖拽画布上的节点可重新定位布料形状；双击画布会释放所有固定点。
- 支持多点触控拖拽，可在触屏设备上同时编辑多处节点。

## 相关文件
- `index.html` —— 包含弹簧求解、渲染循环与控制面板。
- `meta.json` —— 模块元数据，供首页和自动脚本读取。
