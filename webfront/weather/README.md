# 动画天气卡片

> 精美的天气状态卡片，含日出、风、雨、雪等动画效果。

## 功能亮点
- 提供晴、风、雨、雪四类天气动画，以及全局展示模式。
- 卡片展示温度、体感、风速与湿度等信息，配合动效增强氛围。
- 兼顾明暗模式与减弱动画偏好，自动适配不同设备。

## 快速开始
1. 在浏览器中打开 `index.html`。
2. 通过顶部按钮切换关注的天气类型，或使用“全部”查看多张卡片。
3. 点击“暂停”可静止所有动画，再次点击恢复播放。

## 交互提示
- 每张卡片支持无障碍语义标签，可直接用于展示或改造成组件库样式。
- 可通过修改 `cards` 数据结构扩展更多天气类型或数据字段。

## 相关文件
- `index.html` —— 包含布局样式、卡片动画与交互逻辑。
- `meta.json` —— 模块元数据，供首页卡片自动生成使用。
- `prompt.md` —— 设计灵感记录，可作为扩展时的参考草稿。
