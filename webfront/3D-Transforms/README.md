# 3D 图形变换

> 纯 CSS 与 Canvas 的 3D 旋转、透视与光影效果。

## 功能亮点
- 通过滑块控制观察角度、透视深度与动画速度，实时预览空间变换。
- 提供立方体、环轨、多面体等多种展示模式，可单独或组合播放。
- 支持拖拽旋转、滚轮缩放与一键重置视角，适合演示 3D 变换基础。

## 快速开始
1. 在浏览器中打开 `index.html`。
2. 使用控制面板调整角度、透视或切换展示模式。
3. 在舞台区域拖拽可自由旋转视角，双击恢复默认状态。

## 交互提示
- 暂停动画后可手动旋转观察细节，配合透视滑块分析投影效果。
- 在移动端使用双指缩放与拖动同样适用，便于演示交互操作。

## 相关文件
- `index.html` —— 集成样式、交互逻辑与 Canvas 绘制。
- `meta.json` —— 模块元数据，供首页与自动化脚本读取。
