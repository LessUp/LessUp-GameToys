# 规则六边形中的小球自由落体

## 快速上手
- **打开方式**：直接用浏览器打开 `index.html`，无需构建或额外依赖。

## 功能概览
- **规则六边形舞台**：画布中央生成自适应的规则六边形边界。
- **小球物理**：小球受重力影响下落，与边界及彼此发生弹性碰撞。
- **实时反馈**：顶部状态条实时展示小球数、重力、弹性系数与 FPS。
- **可调参数**：滑块与数字输入支持调整重力、弹性、小球数量。
- **交互控制**：提供“暂停/继续”“重置”按钮，便于演示与回放。

## 技术要点
- **单文件实现**：纯原生 HTML/CSS/JavaScript，保持 KISS 原则。
- **高 DPI 支持**：根据设备像素比缩放 `canvas`，保证清晰度。
- **基于时间步长的更新**：使用 `requestAnimationFrame` 与 delta time，确保不同刷新率下表现一致。
- **碰撞处理**：
  - 六边形边界采用最近点检测并做位置修正与速度反射。
  - 小球间采用等质量完全弹性碰撞，交换法线分量并保持切向速度。

## 文件结构
- **`index.html`**：主页面，包含样式、逻辑与 UI。
- **`prompt.md`**：需求描述（方便回溯设计动机）。
- **`README.md`**（当前文件）：使用与实现说明。
