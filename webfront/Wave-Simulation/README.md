# 波纹与流体模拟

> 交互式水波传播与流体动力学可视化。

## 功能亮点
- 通过阻尼、传播速度、扰动概率与强度等参数控制波动方程的行为。
- 内置水面、法线、速度、深度等多种着色模式，可从不同角度观察波场。
- 面板按钮支持一键生成中心水滴或静止水面，便于对比不同状态。

## 快速开始
1. 在浏览器中打开 `index.html`。
2. 调整阻尼与传播速度滑块，观察波纹扩散速度与衰减程度的变化。
3. 使用“中心水滴”快速注入扰动，或点击画布任意位置自行添加波纹。

## 交互提示
- 拖拽或长按画布可持续注入扰动，按住 Shift 将产生反向凹陷。
- 降低分辨率可以换取更高帧率，适合在移动设备上保持流畅体验。

## 相关文件
- `index.html` —— 包含波动模拟、渲染与控制面板逻辑。
- `meta.json` —— 模块元数据，供自动化脚本与首页使用。
