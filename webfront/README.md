# webfront 前端可视化示例

单文件 HTML/CSS/JS 实现的交互式前端小项目，无需构建工具，直接在浏览器打开即可体验。

## 目录结构

- `Cube-Simulation/`：3×3 魔方模拟器，支持相机控制和层旋转
- `Free-Fall-of-Small-Balls-in-a-Hexagon/`：Canvas 物理引擎演示，小球在六边形内自由下落
- `weather/`：CSS 动画天气状态卡片
- `snake/`：霓虹风格贪吃蛇游戏，支持键盘与触摸操作
- `2048/`：经典 2048 数字合并游戏

## 使用方式

进入对应子目录，直接在浏览器中打开 `index.html` 即可体验。

## 设计原则

- 保持单文件结构，无外部依赖
- 使用原生 HTML/CSS/JavaScript
- 追求简洁、可读、可复用的代码风格
- 支持桌面与移动端响应式设计
