# 形状变形动画

> 几何图形的平滑变形与过渡动画。

## 功能亮点
- 通过参数插值在圆形、星形、心形等轮廓间平滑切换。
- 可调节变形速度、路径细分、噪声扰动、旋转速度与发光强度。
- 支持多种形状序列与渐变调色板，并可随机生成配色方案。

## 快速开始
1. 在浏览器中打开 `index.html`。
2. 调整控制面板的滑块或下拉菜单，观察形状随参数变化的动态效果。
3. 使用“下一形状”或点击画布锁定当前帧，便于逐步分析过渡细节。

## 交互提示
- 滚轮缩放可放大局部细节，空格键临时暂停动画。
- “随机颜色”按钮会重绘调色板，适合探索不同配色组合。

## 相关文件
- `index.html` —— 形状插值逻辑、控制面板与渲染循环集中于此。
- `meta.json` —— 模块元数据，供首页卡片与自动化脚本读取。
