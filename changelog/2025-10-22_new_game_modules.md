# 2025-10-22 新增终端小游戏与网页交互模块

## 新增
- `terminal-dance/snake/`：贪吃蛇小游戏，提供 Go 与 Rust 实现，支持 ASCII / Emoji 渲染、WASD 控制、速度可调。
- `terminal-dance/life/`：康威生命游戏，提供 Go 与 Rust 实现，支持自定义世界尺寸、迭代速度与初始密度。
- `webfront/2048/`：纯前端 2048 游戏，包含手势/键盘控制、得分统计与结束提示。
- `webfront/snake/`：Canvas 版霓虹贪吃蛇，包含触摸虚拟按键、本地最高分记录与音效反馈。
