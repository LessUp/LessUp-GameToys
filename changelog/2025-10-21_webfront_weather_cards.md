# 2025-10-21 Webfront: 单文件动画天气卡片

- 位置：`CodingToys/webfront/index.html`
- 特性：
  - 深色主题、并排展示 4 张天气卡片（晴/风/雨/雪）。
  - CSS 动效：日光旋转光束、风线与摇曳树、雨滴坠落与水坑涟漪、雪花飘落与积雪。
  - JavaScript 切换：按钮切换聚焦天气、全局暂停/继续。
  - 单文件打包（HTML/CSS/JS），KISS。
- 交互：
  - 顶部按钮可切换 `全部/晴/风/雨/雪`，非聚焦卡片降低饱和度并暂停动画。
  - `暂停` 按钮支持全局暂停/恢复。
- 备注：
  - 遵循用户规则：中文输出、记录变更、KISS。
