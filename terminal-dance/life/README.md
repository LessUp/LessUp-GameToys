# Life 康威生命游戏

终端版康威生命游戏，支持 Go 与 Rust 双实现，可调节演化速度、世界尺寸与初始细胞密度。

## 特性
- ASCII 与 Emoji 双渲染风格
- 周期性边界（世界两侧相连）
- 自定义世界宽度、高度、演化速度
- 可设置初始细胞密度，快速生成不同模式

## 运行示例

### Go 版本
```bash
cd go
go run ./ -emoji=false -speed=100 -w=40 -h=20 -density=0.3
```

### Rust 版本
```bash
cd rust
cargo run --release -- --emoji=false --speed=100 --w=40 --h=20 --density=0.3
```

## 参数说明
- `-emoji` / `--emoji`：是否使用 Emoji 渲染（默认 false）
- `-speed` / `--speed`：每帧延迟毫秒数（默认 100）
- `-w` / `--w`：世界宽度（默认 40）
- `-h` / `--h`：世界高度（默认 20）
- `-density` / `--density`：初始细胞密度 0.0-1.0（默认 0.3）

## 交互
- 演化自动进行，可随时使用 `Ctrl+C` 退出，程序会恢复光标显示。

## 说明
- 渲染使用 ANSI 控制序列，在大多数终端中表现良好。
- 生命游戏采用循环边界，细胞从最右侧离开会在最左侧出现，从底部离开会在顶部出现。
