# Snake 贪吃蛇

经典贪吃蛇游戏的终端实现。使用 WASD 或方向键控制蛇的移动，吃到食物增长身体并获得分数。

## 特性
- ASCII 与 Emoji 双渲染风格
- 可自定义游戏区域尺寸
- 可调节游戏速度
- 游戏结束后显示最终得分

## 运行示例

### Go 版本
```bash
cd go
go run ./ -emoji=false -speed=150 -w=20 -h=10
```

### Rust 版本
```bash
cd rust
cargo run --release -- --emoji=false --speed=150 --w=20 --h=10
```

## 参数说明
- `-emoji` / `--emoji`：是否使用 Emoji 风格（默认 false）
- `-speed` / `--speed`：每帧延迟毫秒数（默认 150，数值越小越快）
- `-w` / `--w`：游戏宽度（默认 20）
- `-h` / `--h`：游戏高度（默认 10）

## 操作说明
- `W` / `↑`：向上
- `S` / `↓`：向下
- `A` / `←`：向左
- `D` / `→`：向右
- `Q`：退出游戏
- `Ctrl+C`：强制退出

## 游戏规则
- 撞墙或撞到自己会导致游戏结束
- 吃到食物会增长 1 格身体并获得 1 分
- 目标是获得尽可能高的分数
