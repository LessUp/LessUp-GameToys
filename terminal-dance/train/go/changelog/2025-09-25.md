# 2025-09-25 项目初始化与小火车动画

- 新建 Go 项目并遵循 KISS 原则：无外部依赖，直接可运行。
- 添加命令行参数：`--cars`（车厢数）、`--speed`（帧率）。
- 使用 ANSI 转义在单行进行动画渲染，支持优雅退出与光标恢复。

## 变更明细
- 新增文件：
  - `go.mod`
  - `main.go`
  - `README.md`
  - `changelog/2025-09-25.md`
- 更新 `main.go`：增加 `--emoji` 参数、表情小火车动画与多字节字符截断支持
- 更新 `README.md`：记录新参数

## 使用示例
```bash
go run . --cars 4 --speed 15
```
