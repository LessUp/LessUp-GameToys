# 终端小火车 (Go)

一个极简、无外部依赖的 Go 命令行小工具，在终端里显示一列会“跑”的小火车。遵循 KISS 原则，只使用 ANSI 转义控制刷新，支持自定义车厢数量与帧率。

## 特性
- 简单稳定：仅标准库与 ANSI 转义序列
- 可配置：`--cars` 调整车厢数，`--speed` 调整帧率，`--emoji` 切换表情风格
- 友好退出：`Ctrl+C` 终止后自动恢复光标

## 快速开始

```bash
# 运行（无需安装依赖）
go run . --cars 4 --speed 15

# 或编译为可执行文件
go build -o train
./train --cars 6 --speed 12
```

## 参数说明
- `--cars` 车厢数量，范围 0-20，默认 3
- `--speed` 帧率（FPS），范围 1-60，默认 12
- `--emoji` 是否显示为表情小火车，默认关闭

## 使用提示
- 若终端不支持 ANSI 转义，动画可能无法正常显示。
- 若中断后屏幕显示异常，可执行 `reset` 或清屏后重试。
- 默认宽度来自环境变量 `COLUMNS`，不可用时退回 80 列。

## 实现概要
- 使用 `flag` 解析命令行参数
- 使用 `time.Ticker` 以固定 FPS 驱动帧刷新
- 通过 `\r` 与 `ESC[2K` 擦除并重绘同一行
- 使用 `SIGINT`/`SIGTERM` 捕获做优雅退出，恢复光标显示
